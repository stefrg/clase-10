Hola Git

git init: una sola vez por proyecto

git status: muestra el estado actual de nuestro proyecto

git add: podemos guardar con este comando todos los o el archivo que necesitamos por ejemplo git add . guarda todos los archivos modificados, agregados o eliminados

git commit: le damos titulo y descripcion al commit que hacemos para despues identificar que se hizo, por ejemplo git commit -m "el -m significa mensaje, y entre las comillas va la descripcion de lo que hicimos". Lo ideal es trabajar los commit por bloques, no dejar pasar mucho tiempo ni hacer cambios muy pequenos

git log: nos muestra el historial de commits, junto al autor, fecha y hora en el que se realizo
